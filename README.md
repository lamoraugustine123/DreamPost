# DreamPost - Share Your Dreams with the World

A beautiful, modern web application for creating, sharing, and exploring dreams, fantasies, and creative stories with enterprise-grade security.

## ✨ Features

### 🔐 Security & Authentication
- **Two-Factor Authentication (2FA)** with TOTP support
- **Account Lockout Protection** against brute force attacks
- **Session Management** with device tracking
- **Security Audit Logging** with detailed event tracking
- **Enhanced Password Requirements** with strength validation

### 📝 Content Creation
- **Multiple Content Types**: Dreams, Fantasies, Scenarios, Stories
- **Advanced Story Elements**: Characters, themes, settings
- **Rich Metadata**: Story length, target audience, writing style
- **Image Upload Support** with preview functionality
- **Auto-save Drafts** for content protection

### 🎨 User Interface
- **Glass-morphism Design** with carbon-fiber aesthetic
- **Responsive Layout** for all devices
- **Dark/Light Theme** support
- **Customizable Accent Colors**
- **Smooth Animations** and micro-interactions

### 👥 Social Features
- **Like & React** to content
- **Comment System** with threaded discussions
- **Bookmark/Save** favorite content
- **Follow Other Users**
- **Trending & Recommended** content discovery

### ⚙️ Comprehensive Settings
- **Account Management**: Profile, email, password
- **Privacy Controls**: Data sharing, blocked users
- **Notification Preferences**: Email and push notifications
- **Appearance Settings**: Theme, colors, display options
- **Content Filtering**: Mature content, language preferences
- **Advanced Options**: Performance, developer tools

## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation
```bash
# Clone repository
git clone https://github.com/yourusername/dreampost.git

# Navigate to project
cd dreampost

# Install dependencies
npm install

# Start development server
npm start
```

### Development
```bash
# Start with auto-reload
npm run dev

# Run tests
npm test

# Build for production
npm run build
```
